Welcome To Buyut's Log

Maaf ya kalau berantakan :)

Setelah berulangkali kerepotan dengan web hosting dan sewa domain yang lupa atau tak terbayar, dan pada akhirnya harus daftar ulang, memindahkan konten ke domain dan hosting baru, akhirnya saya pun menjadi malas menulis. Tapi waktu mengubah seseorang. Adanya engine Ghost On Github Pages yang membuat publikasi dapat dilakukan lokal kapan saja - dan nantinya di-push ke Github GH-Pages repository, sebagaimana repo lainnya - kembali merangsang saya untuk menulis.
Kenapa Github? Karena saya ingin menulis mengenai tool-tool open source yang kira-kira bermanfaat dan belum terlalu banyak dijumpai dalam Bahasa Indonesia. Jadi, Anda sudah dapat menyimpulkan bahwa blog ini berisi hal-hal teknis, khususnya teknologi informasi, lebih khusus lagi: Pemrograman.

Dasar Pikiran:

  1. Saya menulis bukan karena saya pintar. Tapi karena saya sedang belajar. Ini sesuai pepatah populer yang mengatakan, "Pintar, membuat. Bodoh, mengajar." Maksudnya kurang lebih: Orang yang telah pintar dengan suatu hal cenderung berbuat dan berkreasi dengan sesuatu itu, sementara yang masih awam atau baru mempelajarinya justru menulis dan mengajar ke orang lain.
  2. Bahwasannya tulisan-tulisan saya nantinya adalah catatan tentang perkenalan saya dan pembelajaran saya akan sesuatu, bukan referensi dan panduan teknis, karena saya menulisnya saat saya mempelajarinya, di mana saya dapat mengambil screenshot dan membuat catatan atas apa yang saya alami dan mungkin juga menambahkan pemikiran dan ide yang timbul di pikiran saya.
  3. Perlu diketahui bahwa kemungkinan besar saya akan menulis lebih banyak tentang Tabris JS, sebuah JavaScript framework yang sangat membantu membangun aplikasi native di platform Android, iOS dan Windows Mobile dalam bahasa JavaScript, yang kemudian di-bridge menggunakan Cordova. Tabris JS tidak menggunkan embedded web browser sebagaimana aplikasi hybrid seperti Onsen UI, Ionic dan sejenisnya. Tabris JS lebih mirip dengan React Native dan NativeScript. Kita akan membahas tentang Tabris JS lebih banyak nanti.

Salam hangat,
Buyut.
